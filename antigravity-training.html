Antigravity

IT 동아리 Vibe 코딩 특강

"코드를 외우는 것이 아니라, 상상하고 논리적으로 설명하는 시대입니다."

구글 안티그래비티(Antigravity) 시작하기

이 섹션에서는 안티그래비티가 무엇인지, 기존 코드 에디터와 어떻게 다른지 알아봅니다. 안티그래비티는 타자기가 아닌 여러분의 **전속 수석 개발자**입니다. 복잡한 문법을 몰라도, 아이디어만 있다면 AI 에이전트가 코드를 기획하고 작성해 줍니다. 아래 3가지 핵심 인터페이스를 클릭하여 각 역할을 확인해 보세요.

💬

에이전트 매니저

AI에게 명령을 내리고 소통하는 채팅창. 단축키 Ctrl+E

이곳에서 "웹앱 만들어줘"와 같은 프롬프트를 입력합니다. 에러가 났을 때 로그를 붙여넣고 해결을 요청할 수도 있습니다.
💻

코드 에디터

AI가 작성한 코드가 실시간으로 찍히는 캔버스 공간.

작성된 코드를 직접 눈으로 확인하고, 필요하다면 일반 에디터처럼 직접 코드를 타이핑하여 수정할 수도 있습니다.
🎨

아티팩트

AI의 작업 결과물, 시안, 미리보기를 직관적으로 확인.

복잡한 텍스트 로그 대신, 완성된 화면의 렌더링 결과나 디자인 시안을 바로 보여주어 빠른 피드백이 가능합니다.

마법의 프롬프트 가이드 (RTDS)

안티그래비티의 핵심은 코딩 실력이 아닌 '대화의 기술'입니다. 이 섹션에서는 AI에게 지시를 내리는 완벽한 4단계 공식인 RTDS를 배우고, 나쁜 프롬프트가 좋은 프롬프트로 어떻게 변화하는지 직접 인터랙션하며 비교해 볼 수 있습니다.

실전 연습: 동아리 아이디어 게시판

"동아리 애들이 쓸 게시판 하나 만들어줘. 예쁘게 해줘."
결과: AI가 의도를 파악하지 못하고 촌스러운 기본 스타일의 게시판을 만들 확률 99%

Role너는 10년 차 트렌디한 프론트엔드 수석 개발자야.

Task우리 IT 동아리원들이 아이디어를 자유롭게 올릴 수 있는 '아이디어 공유 웹앱'을 만들어줘.

StackReact와 Tailwind CSS를 사용해서 단일 페이지(SPA)로 구성해 줘.

Details 1. 디자인: 다크 모드 테마, 주요 버튼은 네온 퍼플(#b026ff) 포인트, 애플 웹사이트처럼 여백이 많게.
2. 기능: 상단 큰 제목, 텍스트 박스와 등록 버튼, 카드 형태의 아이디어 나열.
3. 인터랙션: 등록 버튼 클릭 시 부드러운 애니메이션 효과.

실전 티키타카 워크플로우

첫 프롬프트를 작성했다면, 이제 AI와 소통하며 앱을 완성해 나가는 과정입니다. 이 섹션은 앱 개발의 3단계 순환 과정을 보여줍니다. 각 단계를 클릭하여 구체적인 행동 강령과 디버깅 대처법을 확인하세요.

1

첫 초안 확인하기

AI가 코드를 작성한 후 '아티팩트(Artifacts)' 창에 결과물을 띄워줍니다. 미리보기 화면을 꼼꼼히 살펴보고 마음에 드는 부분과 수정이 필요한 부분을 찾습니다.
2

수정 요청하기 (Iterative Prompting)

코드를 직접 고칠 필요가 없습니다. 말로 지시하세요.

  • "아이디어 카드 모서리가 너무 각져 있어. 둥글게(border-radius) 처리해 줘."
  • "모바일 화면 크기로 줄였을 때 글씨가 너무 커서 깨져. 반응형으로 수정해 줘."
3

에러(Error) 대처법

빨간색 에러 창이 떠도 당황하지 마세요!

해결법: 에러 메시지를 복사해서 에이전트 매니저에 붙여넣습니다.
"터미널에 Module not found 에러가 나. 원인을 찾아서 고쳐줘."
그러면 AI가 스스로 패키지를 설치하거나 코드를 수정하여 디버깅을 완료합니다.

웹앱 배포 플랫폼 비교

로컬에서 완성된 동아리 웹앱을 친구들과 공유하기 위한 무료 배포 플랫폼을 비교합니다. 프로젝트의 성격(React 기반, 정적 사이트, DB 필요 여부)에 따라 가장 적합한 플랫폼을 선택할 수 있도록 주요 지표를 시각화했습니다. AI에게 "Vercel에 배포하게 세팅해줘"라고 말하면 모든 준비를 알아서 해줍니다.

Vercel (버셀) 강력 추천

React, Next.js 등 최신 프론트엔드 프레임워크에 최적화. 깃허브 연동 자동 배포.

GitHub Pages

가장 근본적이고 직관적인 방법. HTML/CSS/JS 중심의 가벼운 정적 웹사이트에 적합.

Firebase Hosting

구글 서비스. 향후 로그인이나 데이터베이스(DB) 저장이 필요한 앱 확장에 매우 유리.
